# Memory LUV Kit — Configurator

**A LUVNOTES product by RMS Omni Media**

A web-based configurator that generates a personalized, self-updating orientation display for people living with memory challenges, cognitive change, or extended recovery.

## What it does

Caregivers fill out a 5-step form and download a custom `.html` file. That file:
- Shows the current time and date in large, readable text
- Pulls live weather automatically by zip code
- Displays the right caregiver name by day and shift
- Triggers full-screen medication reminders with a chime
- Shows today's schedule and upcoming appointments
- Switches automatically between day, evening, and night modes
- Reloads itself at midnight — zero daily maintenance required

## How to use this repo

### Test it locally
1. Clone or download this repo
2. Open `index.html` in Chrome or Edge
3. Fill out the form and click Download
4. Open the downloaded file in your browser — press F11 for full screen

### Deploy to GitHub Pages
1. Push this repo to GitHub
2. Go to Settings → Pages
3. Set source to `main` branch, root folder
4. Your configurator will be live at `https://yourusername.github.io/memory-luv-kit/`
